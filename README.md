# 📊 Donations Dashboard 💵
An AI-powered analytics dashboard for small nonprofits, built with **Streamlit** and the **Claude API**.

Built for small non-profits without dedicated staff to analyse their donations data! Answer important questions such as "Is our donor base growing?" and "Which campaign should we repeat?" without having to dig.

Easily upload a donations CSV and receive interactive visualizations! In addition, utilize the ai-powered tools: executive briefing and data chat bot for additional questions.

## ✨ Features
 
- **Key metrics at a glance** — total raised, donation count, unique donors, average gift
- **Interactive Plotly charts** — monthly giving trends, top campaigns, payment method breakdown, and a top-donors table with inline progress bars
- **🤖 AI Executive Briefing** — Claude analyzes the data and writes a plain-English briefing with key trends, risks, and a concrete recommendation
- **💬 Sidebar Data Chat** — a conversational chatbot (with memory) that answers ad-hoc questions like *"What happened in October?"* grounded in the actual numbers
- **🎨 Theme selector** — three color palettes that restyle every chart instantly
- **Works out of the box** — bundled sample datasets, or upload your own CSV

## 🛠️ Tech Stack
 
Python · Streamlit · pandas · Plotly · Anthropic Claude API
